securerandom Gem

Description

Provides cryptographical randomness from openssl or the OS

RubyGems · GitHub · RDoc

Info

  • This standard library is a default gem
  • Use require "securerandom" to load this library
  • The library is written in Ruby, there are no native extensions
  • Current maintainer: Tanaka Akira (akr)
