rubygems Gem

Description

RubyGems allows you to download, install, and use Ruby software packages on your system

RubyGems · GitHub · RDoc

Info

  • This standard library is a default gem
  • The library is required automatically
  • The library is written in Ruby, there are no native extensions
  • Current maintainers: Eric Hodel (drbrain), Hiroshi SHIBATA (hsbt)
