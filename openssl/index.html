openssl Gem

Description

Wraps OpenSSL for cryptographic functionality

RubyGems · GitHub · RDoc

Info

  • This standard library is a default gem
  • Use require "openssl" to load this library
  • The library contains native extensions
  • Current maintainer: Kazuki Yamaguchi (rhe)
