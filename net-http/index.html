net-http Gem

Description

Support for HTTP (Hypertext Transfer Protocol)

RubyGems · GitHub · RDoc

Info

  • This standard library is a default gem
  • Use require "net/http" to load this library
  • The library is written in Ruby, there are no native extensions
  • Current maintainer: NARUSE, Yui (naruse)
  • Similar gems and alternatives: HTTP, excon, faraday, httparty, httpclient, em-http-request
