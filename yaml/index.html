yaml Gem

Description

require 'yaml' will load the psych default gem, yaml/dbm is a wrapper around DBM using YAML serialization, and yaml/store lets you use pstore with YAML

RubyGems · GitHub · RDoc (1) · RDoc (2) · RDoc (3)

Info

  • This standard library is a default gem
  • Use require "yaml" to load this library
  • The library is written in Ruby, there are no native extensions
  • Current maintainers: Aaron Patterson (tenderlove), Hiroshi SHIBATA (hsbt)
